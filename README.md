# Бэкэнд с БД на Sqlite

## Создание БД

Создать пустую локальную БД:

```bash
make db.recreate
```

Добавить в таблицу с новостями фейк-новости:

```bash
make db.fake.save
```

Удалить все из БД:

```bash
make clean
```

## Запуск сервера

В `weedly/app.py` создается приложение.
Сервер запускается из `__main__.py` командой `make run`.
